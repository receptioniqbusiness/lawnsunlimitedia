# Lawns Unlimited of Iowa - Vercel/GitHub Ready Site

This version is packaged for a normal GitHub + Vercel workflow.

## Local development

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in the terminal.

## Production build

```bash
npm run build
npm run preview
```

## Deploy to GitHub + Vercel

1. Create a GitHub repo.
2. Upload all files from this folder to the repo root.
3. Import the repo into Vercel.
4. Framework preset: **Vite**.
5. Build command: `npm run build`
6. Output directory: `dist`

## Notes

- This is a static marketing site built as a multi-page Vite project.
- Replace the temporary logo in `assets/icons/logo-mark.svg` when you have the final logo.
- Update any placeholder email/contact destinations before launch.
