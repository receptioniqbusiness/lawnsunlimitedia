## Windows CMD preview
1. Extract the ZIP anywhere on your computer.
2. Open **Command Prompt**.
3. Run:
   cd /d C:\path\to\lawnsunlimited-improved
4. Start a local server:
   py -m http.server 8000
5. Open:
   http://localhost:8000

## Optional quick-start file
You can also double-click `preview-site.cmd` after extraction if Python is installed and on your PATH.
